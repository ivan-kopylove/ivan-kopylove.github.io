Java is also an engine (like a game engine).

Unity / Unreal engine is a next step of generalization of a elements of games - they have features shared between all games in general - physics, levels, etc.

Some features are used very often, so that it makes sense to generalize them. By generalizing a lot of such things a standard library appear.

Standard library is a standard toolset

| tool      | std lib                                    |
| --------- | ------------------------------------------ |
| photoshop | layers, filters, crop, select, etc         |
| java      | standard functions - append, max, min, len |
| unity     | physics, scenes, levels, graphics          |

By combination of using of standard functions of an engine we achieve a result.

Photoshop is generalization of instruments needed to edit images - crop, selection, filters, layers. By combination of tools engine provides we achieve a required result.

Generalization stops at the required level of abstraction. For example, `java` stops at operating system level. Though, there are some native functions that continue their way down.

Usually develops of platform know the statistics and optimize top used functions
